# official-hkjc-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存储，不针对任何具体域名或网站，仅作为通用页面集合进行维护。

## 项目简介

这是一个以 HTML 页面归档为主要目的的仓库。仓库内包含多个自包含的 HTML 文件，可用于展示、测试或参考。所有页面均为独立文件，不依赖外部数据库或后端服务。

## 目录说明

```
.
├── index.html          # 首页 / 导航页
├── pages/              # 归档 HTML 页面目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库入口页，提供页面索引。
- `pages/`：存放所有归档的 HTML 页面。每个文件均为独立完整页面。
- `README.md`：仓库说明文档。

## 页面归档说明

- 所有页面均为静态 HTML 文件，可直接在浏览器中打开。
- 页面内容与样式均内嵌或通过相对路径引用，不依赖外部资源。
- 归档页面不包含任何跟踪、广告或第三方脚本。
- 页面命名遵循 `page-数字.html` 格式，便于索引与排序。

## 使用方式

1. 克隆本仓库：
   ```bash
   git clone https://github.com/your-username/official-hkjc-lab.git
   ```
2. 直接在本地浏览器中打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务直接部署本仓库。
